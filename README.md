# Integration of SysML models in 3D

Eclipse Capella is an open-source Model-Based Systems Engineering (MBSE) tool designed to help engineers and architects model, design, and manage system, software, and hardware architectures effectively. It is built on top of the Arcadia methodology. OpenMBEE (Open Model Based Engineering Environment) is an open source collaborative engineering system. ATL and Acceleo based transformation from SysML to Modelica for Papyrus Neon. The Parallel Object-Oriented Specification Language (POOSL). OML Rosetta an Eclipse RCP (and plugin). Virtual Satellite is a DLR open source software for model based systems engineering (MBSE). XMCDA is a data standard which allows to represent MultiCriteria Decision Analysis (MCDA) data elements in XML according to a clearly defined grammar. Apache Commons RDF, Jelly, JXPath and SCXML. Smooks is a framework for building event-driven applications. jOOX stands for Object Oriented XML. db4o database for objects. Advanced Simulation Library (ASL). Chrono is an open-source multi-physics simulation framework. NVIDIA PhysicsNeMo is an open-source deep-learning framework for building, training, fine-tuning, and inferring Physics AI models. OSMO is a platform for scaling complex Physical AI workloads. Modelica3D is a lightweight 3D-visualisation library. Open Source AADL Tool Environment (OSATE). View Editor an extensible, multi-environment view and widget UI for querying and transcluding model elements for Systems Engineering documents. ReqTool is a model and a requirements editor for the Eclipse Capra framework. Modelica library for interactive simulation. This software provides the user a tool (sdf2modelica) and a C++ library to convert 3D rigid body mechanical systems described by the Gazebo Simulator's Simulation Description Format files to Modelica models that use the Mechanics MultiBody library. Simulation framework for nonsmooth dynamical systems. Convert VRML or X3D models to XML3D. XMLUnit is a library that supports testing XML output

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
